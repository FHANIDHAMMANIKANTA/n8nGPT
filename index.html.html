n8nGPT Workflow Automation Platform

n8nGPT is a fair-code licensed workflow automation tool that enables you to connect any app with an API. It's designed to be extensible, flexible, and easy to use for both technical and non-technical users.

Open-source core Self-hostable 300+ integrations

Core Features

Visual Workflow Builder

Drag-and-drop interface to create complex workflows with conditional logic, loops, and error handling.

  • Node-based workflow design
  • Branching and merging
  • Error handling workflows

Trigger System

Multiple ways to initiate workflows based on events, schedules, or manual triggers.

  • Webhook triggers
  • Cron schedules
  • Manual execution
  • Event-based triggers

Data Transformation

Powerful tools to manipulate and transform data between nodes.

  • JSON manipulation
  • Date/string formatting
  • Data aggregation
  • Custom JavaScript/TypeScript

Error Handling

Robust error management with customizable recovery workflows.

  • Automatic retries
  • Error tracking
  • Custom error workflows
  • Notification system

DevOps Capabilities

Enterprise-grade features for professional deployment and management.

  • Environment variables
  • Workflow versioning
